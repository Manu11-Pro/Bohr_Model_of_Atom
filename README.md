# Bohr_Model_of_Atom
This is a repository for a python project using viser for rendering a "Bohr_Model_of_Atom" 

This project's goal is to make learning the Bohr_Model_of_Atom easier and fun by a way to view the 2D rings in a 3D and interactive way!
